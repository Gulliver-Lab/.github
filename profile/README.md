## Repository list

This organization hosts repositories related to the lab's research activities and technical infrastructure.

### Swift

Repositories related to our cluster (Swift)

* [`swift-config`](https://github.com/Gulliver-Lab/swift-config): The NixOS configuration of the cluster
* [`swift-monitoring`](https://github.com/Gulliver-Lab/swift-monitoring): Scripts to monitor activity and usage on the cluster

### IT

Internal tools, automation scripts, system administration, and lab infrastructure.

* `seminar`
* `people`

### Data & Analysis

* [`TIRF`](https://github.com/Gulliver-Lab/TIRF): Package to run particle detection and tracking on microscopy images.
* `DNA`
